# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public issue.** Instead, contact the maintainer privately:

- Open a [GitHub Security Advisory](https://github.com/davidegarbi/openclaw-healthconnect-bridge/security/advisories/new) (preferred)
- Or reach out via GitHub profile: [davidegarbi](https://github.com/davidegarbi)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 72 hours and work on a fix as quickly as possible.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

This policy covers the Android application code in this repository. Issues with Health Connect itself or third-party server endpoints are out of scope.
